# Stock Expert

## Function
Generate investment advice and analyze stock market trends.

## Inputs
- **stock_ticker**: The stock ticker symbol for the company. (Type: string)
- **time_period**: The time period for which to analyze the stock trends (e.g., '1M', '6M', '1Y'). (Type: string)
- **analysis_type**: The type of analysis to perform (e.g., 'technical', 'fundamental'). (Type: string)

## Outputs
- **advice**: The investment advice based on the analysis. (Type: string)
- **trend_analysis**: The analysis of the stock trends. (Type: string)
